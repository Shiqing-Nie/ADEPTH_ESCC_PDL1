# ADEPTH_ESCC_PDL1
Neoadjuvant anti-PD-L1 immunotherapy combined with chemotherapy in locally advanced esophageal squamous cell carcinoma: a multicenter II trial (ADEPTH)

QC_harmony.R is the workflow for processing single-cell data using Seurat. epithelial.R is the workflow for identifying malignant epithelial cells. pseudotime_trajectory.R is the workflow for pseudotime trajectory analysis. cellchat.R is the workflow for cell–cell communication analysis. sctcr.R is the analysis pipeline for scTCR data. The scRNA_scripts folder contains a set of utility functions.
